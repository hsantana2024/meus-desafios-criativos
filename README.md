# meus-desafios-criativos
Desafio Criativo: Extraindo Insights do Feedback de Clientes Bancários
Atue como analista de dados e experiência do cliente no setor bancário.
Sua tarefa é analisar feedbacks de clientes relacionados aos produtos e serviços bancários para identificar padrões, tendências, reclamações recorrentes, elogios e oportunidades de melhoria.
Contexto: A análise será utilizada pelas áreas de Experiência do Cliente, Produtos e Atendimento para apoiar decisões de melhoria nos serviços oferecidos pelo banco, aumentar a satisfação dos clientes e reduzir problemas operacionais. O objetivo é transformar comentários individuais em insights claros, organizados e acionáveis.
Dados disponíveis: A base de dados pode conter as seguintes informações:
•	Data do feedback
•	Canal de atendimento (agência, aplicativo, internet banking, chat, telefone, etc.)
•	Texto do comentário
•	Produto ou serviço citado (Pix, cartão de crédito, conta corrente, empréstimo, investimentos, etc.)
•	Nota de satisfação (quando disponível)
Instruções de análise:
1.	Classifique os feedbacks por:
o	Tema principal
o	Sentimento (positivo, negativo ou neutro)
o	Nível de urgência (baixo, médio ou alto)
o	Produto ou serviço mencionado
2.	Identifique:
o	Principais reclamações recorrentes
o	Principais elogios recorrentes
o	Possíveis causas observadas nos comentários
o	Oportunidades de melhoria
o	Tendências relevantes
3.	Apresente evidências:
o	Utilize exemplos curtos extraídos dos comentários fornecidos.
o	Relacione cada insight aos feedbacks que o sustentam.
4.	Recomende ações:
o	Sugira ações práticas e priorizadas para as equipes responsáveis.
o	Indique possíveis impactos esperados para cada recomendação.
Formato da resposta:
1.	Resumo Executivo
o	Até 5 linhas com os principais achados.
2.	Tabela de Insights
Tema	Sentimento	Urgência	Evidência	Impacto	Ação Recomendada
3.	Principais Prioridades
o	Prioridade 1
o	Prioridade 2
o	Prioridade 3
4.	Limitações da Análise
o	Informe eventuais limitações encontradas nos dados.
Restrições:
•	Utilize apenas as informações fornecidas nos feedbacks.
•	Não invente números, causas ou conclusões sem evidências.
•	Não exponha informações pessoais, bancárias ou dados sensíveis dos clientes.
•	Caso existam dados insuficientes para determinada conclusão, informe claramente essa limitação.
•	Utilize linguagem simples, objetiva, profissional e orientada à tomada de decisão.
•	Mantenha imparcialidade na análise, considerando tanto feedbacks positivos quanto negativos.
